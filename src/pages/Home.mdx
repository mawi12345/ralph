# Welcome to Ralph

This is a minimal **MDX** project running on GitHub Pages!

## What is MDX?

MDX allows you to use JSX in your markdown content. You can import components, write markdown, and use React components seamlessly.

### Features

- Write markdown naturally
- Embed React components
- Style with custom components
- Deploy to GitHub Pages

## Example Component

export const Button = ({ children }) => (
  <button style={{
    padding: '10px 20px',
    backgroundColor: '#0066cc',
    color: 'white',
    border: 'none',
    borderRadius: '5px',
    cursor: 'pointer',
    fontSize: '16px',
    margin: '10px 0'
  }}>
    {children}
  </button>
)

Here's a custom button component defined in MDX:

<Button>Click Me!</Button>
