# Entstehung von Verbindungen

Atome verbinden sich mit anderen Atomen, um einen **stabilen Zustand** zu erreichen. Das Ziel ist immer, die äußerste Elektronenschale zu **füllen** - so wie bei den Edelgasen.

## Der Edelgaszustand

Edelgase (8. Hauptgruppe) sind besonders stabil, weil ihre äußerste Schale **voll besetzt** ist:
- Helium: 2 Außenelektronen (1. Schale voll)
- Neon, Argon, etc.: 8 Außenelektronen

> **Alle Atome streben den Edelgaszustand an!**

Um diesen Zustand zu erreichen, können Atome:
- Elektronen **abgeben**
- Elektronen **aufnehmen**
- Elektronen **teilen**

---

# Die drei Bindungsarten

Je nachdem, welche Atome sich verbinden, entstehen unterschiedliche Bindungsarten:

| Bindungsart | Verbindung zwischen | Beispiele |
|-------------|---------------------|-----------|
| **Metallbindung** | Metall + Metall | Bronze, Messing |
| **Ionenbindung** | Metall + Nichtmetall | Kochsalz (NaCl) |
| **Elektronenpaarbindung** | Nichtmetall + Nichtmetall | Wasser (H₂O) |

---

# 1. Die Metallbindung

## Was passiert?

Wenn sich **zwei oder mehr Metalle** verbinden, entsteht eine **Metallbindung**.

Metalle haben nur **wenige Außenelektronen** (1-3) und geben diese leicht ab. Dadurch entstehen:
- **Positiv geladene Metallrümpfe** (Metall-Ionen)
- **Frei bewegliche Elektronen** ("Elektronengas")

```
    ⊕      ⊕      ⊕      ⊕
      e⁻      e⁻      e⁻
    ⊕      ⊕      ⊕      ⊕
      e⁻      e⁻      e⁻
    ⊕      ⊕      ⊕      ⊕

⊕ = Metallrumpf (positiv)
e⁻ = frei bewegliches Elektron
```

## Eigenschaften von Metallen

Die Metallbindung erklärt wichtige Eigenschaften:

| Eigenschaft | Erklärung |
|-------------|-----------|
| **Elektrische Leitfähigkeit** | Die frei beweglichen Elektronen können den Strom leiten |
| **Wärmeleitfähigkeit** | Elektronen transportieren die Wärmeenergie |
| **Verformbarkeit** | Die Metallrümpfe können aneinander vorbeigleiten |
| **Metallischer Glanz** | Elektronen reflektieren das Licht |

## Legierungen

Wenn verschiedene Metalle gemischt werden, entstehen **Legierungen**:
- **Bronze** = Kupfer + Zinn
- **Messing** = Kupfer + Zink
- **Stahl** = Eisen + Kohlenstoff

# 2. Die Ionenbindung

## Was passiert?

Wenn sich ein **Metall** mit einem **Nichtmetall** verbindet, entsteht eine **Ionenbindung**.

### Der Vorgang:

1. Das Metallatom **gibt** seine Außenelektronen **ab**
   → Es wird zu einem **positiv geladenen Ion** (Kation)

2. Das Nichtmetallatom **nimmt** diese Elektronen **auf**
   → Es wird zu einem **negativ geladenen Ion** (Anion)

3. Die **unterschiedlich geladenen Ionen** ziehen sich an

### Beispiel: Natriumchlorid (Kochsalz)

```
    Na                  Cl
   ○○○               ○○○○○○○
  ○ ⊕ ○      +      ○  ⊙  ○
   ○○○               ○○○○○○○
  (11e⁻)              (17e⁻)
  1 Außene⁻          7 Außene⁻
      ↓                  ↓
      │    Elektron     │
      └───────→─────────┘
      ↓                  ↓
    Na⁺                Cl⁻
   Kation              Anion
   (10e⁻)              (18e⁻)
```

## Das Kristallgitter

Die positiven und negativen Ionen ordnen sich regelmäßig in einem **Kristallgitter** an:

```
    Na⁺  Cl⁻  Na⁺  Cl⁻
    Cl⁻  Na⁺  Cl⁻  Na⁺
    Na⁺  Cl⁻  Na⁺  Cl⁻
    Cl⁻  Na⁺  Cl⁻  Na⁺
```

## Eigenschaften von Salzen

| Eigenschaft | Erklärung |
|-------------|-----------|
| **Hoher Schmelzpunkt** | Starke Anziehung zwischen den Ionen |
| **Kristallstruktur** | Regelmäßige Anordnung der Ionen |
| **Spröde** | Wenn Ionen verschoben werden, stoßen sich gleiche Ladungen ab |
| **Leitet Strom (gelöst/geschmolzen)** | Ionen können sich dann frei bewegen |

## Beispiele für Ionenverbindungen

- Natriumchlorid (NaCl) - Kochsalz
- Calciumchlorid (CaCl₂) - Streusalz
- Natriumhydroxid (NaOH) - Natronlauge

# 3. Die Elektronenpaarbindung (Atombindung)

## Was passiert?

Wenn sich **zwei Nichtmetalle** verbinden, entsteht eine **Elektronenpaarbindung** (auch Atombindung oder kovalente Bindung genannt).

Da Nichtmetalle ihre Elektronen nicht abgeben wollen, **teilen** sie sich stattdessen Elektronenpaare.

### Der Vorgang:

1. Zwei Atome nähern sich an
2. Sie bilden ein oder mehrere **gemeinsame Elektronenpaare**
3. Die gemeinsamen Elektronen gehören **beiden Atomen**
4. Beide erreichen so den Edelgaszustand

### Beispiel: Wasserstoffmolekül (H₂)

```
Zwei Wasserstoffatome (je 1 Elektron):

    H  •      •  H
        ↘    ↙
         ••
    H  ••  H     →    H—H

       gemeinsames
       Elektronenpaar
```

Jedes H-Atom hat nun 2 Elektronen in der Hülle → Edelgaszustand (wie Helium)!

### Beispiel: Wassermolekül (H₂O)

```
        H
       /
    O—H

Ein Sauerstoffatom (6 Außenelektronen) teilt sich mit
zwei Wasserstoffatomen je ein Elektronenpaar.

→ O hat 8 Außenelektronen (Edelgaszustand wie Neon)
→ Jedes H hat 2 Elektronen (Edelgaszustand wie Helium)
```

## Einfach-, Doppel- und Dreifachbindungen

Je nachdem, wie viele Elektronenpaare geteilt werden:

| Bindung | Geteilte Elektronenpaare | Symbol | Beispiel |
|---------|--------------------------|--------|----------|
| Einfachbindung | 1 | — | H—H |
| Doppelbindung | 2 | = | O=O |
| Dreifachbindung | 3 | ≡ | N≡N |

## Eigenschaften von Molekülen

| Eigenschaft | Erklärung |
|-------------|-----------|
| **Niedriger Schmelzpunkt** | Schwache Kräfte zwischen Molekülen |
| **Leiten keinen Strom** | Keine frei beweglichen Ladungsträger |
| **Können gasförmig sein** | Kleine Moleküle (H₂, O₂, CO₂) |

## Beispiele für Moleküle

- Wasser (H₂O)
- Kohlendioxid (CO₂)
- Sauerstoff (O₂)
- Stickstoff (N₂)
- Methan (CH₄)

---

# Zusammenfassung

| Bindungsart | Partner | Elektronen werden... | Ergebnis |
|-------------|---------|---------------------|----------|
| **Metallbindung** | Metall + Metall | abgegeben → Elektronengas | Metall/Legierung |
| **Ionenbindung** | Metall + Nichtmetall | übertragen | Salz (Kristallgitter) |
| **Elektronenpaarbindung** | Nichtmetall + Nichtmetall | geteilt | Molekül |

**Merksatz:** Alle Bindungen haben das Ziel, den **Edelgaszustand** zu erreichen!
