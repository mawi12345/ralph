<MSFormsQuestion>
# Warum gehen Atome Verbindungen ein?

1. Um den Edelgaszustand zu erreichen (volle Außenschale)
2. Um schwerer zu werden
3. Um ihre Farbe zu ändern
4. Um radioaktiv zu werden
</MSFormsQuestion>

<MSFormsQuestion>
# Wie können Atome den Edelgaszustand erreichen?

1. Elektronen abgeben, aufnehmen oder teilen
2. Nur durch Aufnehmen von Elektronen
3. Durch Erhitzen
4. Durch Abkühlen
</MSFormsQuestion>

<MSFormsQuestion>
# Welche drei Bindungsarten gibt es?

1. Metallbindung, Ionenbindung und Elektronenpaarbindung
2. Wasserbindung, Luftbindung und Erdbindung
3. Starke, mittlere und schwache Bindung
4. Einfach-, Doppel- und Dreifachbindung
</MSFormsQuestion>

<MSFormsQuestion>
# Was ist eine Metallbindung?

1. Eine Bindung zwischen zwei oder mehr Metallen
2. Eine Bindung zwischen Metall und Nichtmetall
3. Eine Bindung zwischen zwei Nichtmetallen
4. Eine Bindung mit Edelgasen
</MSFormsQuestion>

<MSFormsQuestion>
# Was entsteht bei einer Metallbindung?

1. Positive Metallrümpfe und frei bewegliche Elektronen (Elektronengas)
2. Negative Metallrümpfe und gebundene Elektronen
3. Neutrale Atome ohne Elektronen
4. Ein Kristallgitter aus Ionen
</MSFormsQuestion>

<MSFormsQuestion>
# Warum leiten Metalle elektrischen Strom?

1. Weil die frei beweglichen Elektronen den Strom transportieren
2. Weil sie viele Protonen haben
3. Weil sie warm sind
4. Weil sie glänzen
</MSFormsQuestion>

<MSFormsQuestion>
# Warum leiten Metalle Wärme gut?

1. Weil die frei beweglichen Elektronen die Wärmeenergie transportieren
2. Weil sie eine hohe Dichte haben
3. Weil sie hart sind
4. Weil sie Licht reflektieren
</MSFormsQuestion>

<MSFormsQuestion>
# Was sind Legierungen? Was ist Bronze?

1. Mischungen aus verschiedenen Metallen - Bronze ist Kupfer + Zinn
2. Reine Metalle - Bronze ist reines Gold
3. Salze - Bronze ist ein Salz
4. Moleküle - Bronze ist ein Molekül
</MSFormsQuestion>

<MSFormsQuestion>
# Was ist eine Ionenbindung?

1. Eine Bindung zwischen einem Metall und einem Nichtmetall
2. Eine Bindung zwischen zwei Metallen
3. Eine Bindung zwischen zwei Nichtmetallen
4. Eine Bindung zwischen Edelgasen
</MSFormsQuestion>

<MSFormsQuestion>
# Was passiert bei der Ionenbindung mit den Elektronen?

1. Das Metall gibt Elektronen ab, das Nichtmetall nimmt sie auf
2. Beide Atome geben Elektronen ab
3. Beide Atome nehmen Elektronen auf
4. Die Elektronen werden geteilt
</MSFormsQuestion>

<MSFormsQuestion>
# Was ist ein Ion?

1. Ein elektrisch geladenes Atom
2. Ein neutrales Atom
3. Ein Molekül
4. Ein Elektron
</MSFormsQuestion>

<MSFormsQuestion>
# Was ist ein Kristallgitter?

1. Eine regelmäßige Anordnung von positiven und negativen Ionen
2. Ein ungeordneter Haufen von Atomen
3. Eine Flüssigkeit
4. Ein Gas
</MSFormsQuestion>

<MSFormsQuestion>
# Wie entsteht Kochsalz (NaCl)?

1. Natrium gibt ein Elektron an Chlor ab, es entstehen Na⁺ und Cl⁻
2. Natrium und Chlor teilen sich ein Elektronenpaar
3. Beide geben Elektronen ab
4. Chlor gibt ein Elektron an Natrium ab
</MSFormsQuestion>

<MSFormsQuestion>
# Warum haben Salze einen hohen Schmelzpunkt?

1. Wegen der starken Anziehung zwischen den Ionen
2. Weil sie schwer sind
3. Weil sie aus Metallen bestehen
4. Weil sie radioaktiv sind
</MSFormsQuestion>

<MSFormsQuestion>
# Wann leiten Salze elektrischen Strom?

1. Wenn sie gelöst oder geschmolzen sind
2. Immer
3. Nie
4. Nur wenn sie fest sind
</MSFormsQuestion>

<MSFormsQuestion>
# Was ist eine Elektronenpaarbindung?

1. Eine Bindung zwischen zwei Nichtmetallen, die Elektronenpaare teilen
2. Eine Bindung zwischen zwei Metallen
3. Eine Bindung zwischen Metall und Nichtmetall
4. Eine Bindung mit Edelgasen
</MSFormsQuestion>

<MSFormsQuestion>
# Wie funktioniert die Elektronenpaarbindung?

1. Zwei Atome teilen sich gemeinsame Elektronenpaare
2. Ein Atom gibt alle Elektronen ab
3. Ein Atom nimmt alle Elektronen auf
4. Die Elektronen verschwinden
</MSFormsQuestion>

<MSFormsQuestion>
# Wie entsteht ein Wasserstoffmolekül (H₂)?

1. Zwei H-Atome teilen sich ein gemeinsames Elektronenpaar
2. Ein H-Atom gibt sein Elektron an das andere ab
3. Beide H-Atome geben ihre Elektronen ab
4. Die Elektronen werden zerstört
</MSFormsQuestion>

<MSFormsQuestion>
# Was ist der Unterschied zwischen Einfach-, Doppel- und Dreifachbindung?

1. Die Anzahl der geteilten Elektronenpaare (1, 2 oder 3)
2. Die Stärke der Bindung
3. Die Farbe der Moleküle
4. Die Größe der Atome
</MSFormsQuestion>

<MSFormsQuestion>
# Warum leiten Moleküle keinen elektrischen Strom?

1. Weil es keine frei beweglichen Ladungsträger gibt
2. Weil sie zu klein sind
3. Weil sie aus Nichtmetallen bestehen
4. Weil sie flüssig sind
</MSFormsQuestion>

<MSFormsQuestion>
# Wie unterscheiden sich die drei Bindungsarten bezüglich der Elektronen?

1. Metallbindung: abgegeben, Ionenbindung: übertragen, Elektronenpaarbindung: geteilt
2. Alle drei teilen Elektronen
3. Alle drei geben Elektronen ab
4. Alle drei nehmen Elektronen auf
</MSFormsQuestion>
